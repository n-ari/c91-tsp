# 巡回セールスマン問題

セールスマンが巡回したがってるんだ。(C91の何か)

## 成瀬順の画像・TSPデータについて

公式HPのキャラクターページにある画像をMATLABによってエッジ検出したものがnarusetsp.bmpです。

そこにtspmaker.rbによってランダムに点を配置して生成したのがsrc/in/naruse10000.inです。

引用元 : http://www.kokosake.jp/character/
